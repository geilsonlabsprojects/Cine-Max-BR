Política de Privacidade

Última atualização: Julho de 2026

1. Coleta de Dados

Não coletamos informações de identificação pessoal de nossos usuários, exceto quando fornecidas voluntariamente para fins de suporte técnico.

2. Armazenamento Local

Utilizamos o localStorage do seu navegador para salvar o status de autorização do seu código de acesso, garantindo que você não precise digitá-lo toda vez que abrir o site.

3. Firebase e TMDb

Utilizamos serviços de terceiros como Google Firebase para banco de dados e TMDb para informações de filmes. Recomendamos a leitura das políticas de privacidade desses fornecedores.

4. Segurança

Empregamos medidas de segurança para proteger a integridade dos dados e das chaves de acesso geradas pelo administrador.
